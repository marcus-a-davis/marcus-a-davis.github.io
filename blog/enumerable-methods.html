Using the Map Method

What if you wanted to do something to each element in an array and then return your results as an array? This is a pretty common task in programming but how can you do this in Ruby? The "each" method is very powerful and you could accomplish with the each method like this: 

      my_array = [1,2,3,4]
      new_array = my_array.each {|element| element * 2 }
      return new_array 
        => [2,4,6,8]

However, as the motto suggests, there's more than one way to do this in Ruby. In fact, Ruby has built in a method that does this very task. The "map" method, which is part of the Enumerable mixin, accomplishes the same thing: 

      my_array = [1,2,3,4]
      my_array.map {|element| element * 2 }
        => [2,4,6,8]

It does this by returning a new array with the results of running the code block you provided on every element in the collection you are running it on. "Map" also has a destructive pair "map!" which alters the original collection you entered: 

      my_array = [1,2,3,4]
      my_array.map! {|element| element * 2 }
        => [2,4,6,8]
      my_array
        => [2,4,6,8]

Finally, "map" also has an alias "collect" which is operationally identical to "map": 

      my_array = [1,2,3,4]
      my_array.collect {|element| element * 2 }
        => [2,4,6,8]
